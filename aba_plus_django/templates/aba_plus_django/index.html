ABA⁺

Bayesian Assumption-Based Argumentation.

This website extends ABAplus, an application for visualising ABA frameworks, with support for random variables in the context of BABA (Bayesian Asssumption Based Argumentation). Users can define the individual components of BABA frameworks, and the system visualises the results the resulting framework. Users can select among worlds by defining the truth values of random variables, and visualise the resulting extensions.

    In ABAplus, a BABA⁺ framework is specified as follows:
  • myAsm(a). specifies that a is an assumption;
  • contrary(a, x). specifies that x is the contrary of assumption a;
  • myRV(c). specifies that c is a random variable;
  • myRule(h, [b1,…,bn]). specifies that h <- b1,…,bn is a rule;
